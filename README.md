# 🛒 Olist E-commerce SQL Analysis

## 📌 Project Overview

This project analyzes the Brazilian Olist E-commerce dataset using SQL Server. The objective is to transform raw transactional data into meaningful business insights through data quality checks, exploratory data analysis (EDA), and business-focused SQL queries.

The analysis covers customer behavior, sales performance, product performance, delivery efficiency, seller performance, reviews, and payment trends.



## 📂 Dataset

- **Dataset:** Olist Brazilian E-commerce Public Dataset
- **Source:** Kaggle
- Contains information about:
  - Customers
  - Orders
  - Products
  - Sellers
  - Payments
  - Reviews
  - Geolocation

## 🔍 Key Insights

- Health & Beauty generated the highest revenue.
- São Paulo (SP) generated the highest revenue among all states.
- Revenue showed a sharp decline in September 2018.
- Only a small percentage of customers made repeat purchases.
- Approximately 92% of orders were delivered on or before the estimated delivery date.
- Longer delivery times were associated with lower review scores.
- Most customers preferred paying in one or two installments.
